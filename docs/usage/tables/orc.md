# ORC

This page explains how to work with Avro tables in Unity Catalog.

## Create ORC table

Use the `bin/uc table create` command with the `--format ORC` option to create a new ORC table in Unity Catalog.

Run the following command with the correct `storage_location` to create a new ORC table with 2 colummns: `some_numbers` and `some_letters`:

```
bin/uc table create \
  --format ORC \
  --full_name unity.default.test \
  --columns "some_numbers INT, some_letters STRING" \
  --storage_location $DIRECTORY$
```
